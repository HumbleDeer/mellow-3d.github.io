ssh to your klipper host console - Clone the CanBoot firmware to your Klipper Host ``` cd ~/ git clone https://github.com/Arksine/CanBoot ``` - From your Fluidd or Mainsail web UI Edit Moonraker.conf and add the following at the bottom to allow moonraker to manage updates to the CAN bus software. ``` [update_manager CanBoot] type: git_repo path: ~/CanBoot origin: https://github.com/Arksine/CanBoot.git is_system_service: False ``` ### Compile CanBoot firmware - cd to the Canboot directory ``` cd CanBoot ``` - Run make clean ``` make clean ``` - Open menuconfig ``` make menuconfig ``` - Set the following settings - Micro Controller : STMicroelectronics STM32 - Processor Model: {{page.processor}} - Build: {{page.offset}} - Clock Ref: {{page.clock}} - Comm Interface - {{page.com}} - Application Offset 32 Kib {% if page.firmware == "can" %} - Can bus Speed: 500000 {% endif %} {% include image.html file=cancom_img1 url=cancom_url1 alt=cancom_cap1 caption=cancom_cap1 %} - Quit and save the configuration - Run the make command to compile the firmware ``` make -j4 ``` - You should now have a canboot.bin file at ~/CanBoot/out/ ### Flash CanBoot to the {{page.boardname}} MCU - Install the boot jumper to the Boot0 pin and 3.3v pin as pictured. {% include image.html file=cancom_img2 url=cancom_url2 alt=cancom_cap2 caption=cancom_cap2 %} - Use a USB-C data cable to connect the {{ boardname }} board to the klipper host, Make sure that the dfu jumper is installed before connecting the USB cable. - Run lsusb to see if the connection is successful, copy the USB ID in the blue box. Note that the board is in DFU mode. ``` lsusb ``` {% include image.html file=canburn_img1 url=canburn_url1 alt=canburn_cap1 caption=canburn_cap1 %} - Erase and Burn the firmware , replace 0483:df11 in the following command with the USB ID copied earlier ``` sudo dfu-util -a 0 -D ~/CanBoot/out/canboot.bin --dfuse-address 0x08000000:force:mass-erase:leave -d 0483:df11 ``` - You should get a download progress bar and File downloaded successfuly when the burning is complete. {% include image.html file=canburn_img2 url=canburn_url2 alt=canburn_cap2 caption=canburn_cap2 %} - If the burn was sucessful remove the jumper from the Boot0 pin and power cycle the printer. {% if page.firmware == "can" %} - The tool board is now ready to install [klipper firmware](./fly-super8_klipper_can.html). {% else if page.firmware == "usb" %} - The tool board is now ready to install [klipper firmware](./fly-super8_klipper_usb.html). {% endif %}
